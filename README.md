# Bias Analysis in Recruitment

##  Overview
This project analyzes bias in candidate shortlisting processes using data-driven techniques. It compares traditional (baseline) hiring decisions with fairness-adjusted models to identify and reduce potential discrimination in recruitment.

The dataset (`bias_analysis_final.csv`) includes candidate-level attributes such as gender, college tier, experience, scores, and shortlisting outcomes.

---

##  Objectives
- Identify bias in baseline shortlisting processes  
- Compare results with fairness-aware ranking models  
- Evaluate the impact of bias mitigation techniques  
- Promote equitable and responsible hiring practices  

---

##  Dataset
The dataset contains:
- Candidate demographics (e.g., gender)
- Educational background (college tier)
- Experience (years)
- Assessment scores
- Baseline shortlist decision
- Fair shortlist decision

---

##  Key Features
- Exploratory Data Analysis (EDA)  
- Bias detection across demographic groups  
- Comparison of baseline vs fairness-based selection  
- Insights into hiring disparities  
- Data-driven recommendations for fair recruitment  

---

##  Tech Stack
- Python  
- Pandas  
- NumPy  
- Matplotlib / Seaborn  
- Jupyter Notebook  

---

## Analysis Performed
- Distribution analysis of candidate attributes  
- Shortlisting rate comparison by gender and background  
- Evaluation of fairness adjustments  
- Identification of potential bias patterns
